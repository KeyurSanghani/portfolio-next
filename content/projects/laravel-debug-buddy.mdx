## Laravel Debug Buddy

Laravel Debug Buddy is a powerful VS Code extension specifically designed to enhance the Laravel development experience by providing intelligent debugging tools directly within your development environment. With over **1000+ installations** from the VS Code Marketplace, it has become an essential tool for Laravel developers worldwide.

### ‚ú® Key Features

#### **Intelligent Context-Aware Debugging**
The extension's flagship feature automatically analyzes PHP code context and places debug statements in the correct location:
- **Smart insertion** for control structures (if, foreach, while, etc.)
- **Method chain detection** for fluent Laravel queries
- **Array definition handling** for complex data structures
- **Try-catch block awareness** for error handling contexts
- **Return statement detection** to avoid unreachable code

#### **Instant Logging & Variable Dumping**
- **Quick Log Variable** (`Ctrl+Shift+L`) - Instantly log variables with Laravel's Log facade
- **DD Variable** (`Ctrl+Shift+D`) - Dump and die for immediate debugging
- **Quick Actions** - Log or dump selected code with keyboard shortcuts
- **Smart Variable Detection** - Automatically detects variable names and context

#### **Performance Monitoring**
- **Execution Timing** (`Ctrl+Shift+T`) - Measure code performance with precise timing
- **Memory Usage Tracking** - Monitor resource consumption
- **Performance Insights** - Identify bottlenecks in your Laravel application

#### **Automated Debug Cleanup**
- **Remove Debug Statements** (`Ctrl+Shift+R`) - Clean up all debug code with one command
- **Selective Cleanup** - Remove only specific types of debug statements
- **Production Safety** - Prevent debug code from reaching production environments

### üõ†Ô∏è Technical Implementation

The extension leverages advanced PHP parsing and AST (Abstract Syntax Tree) analysis to understand code context:

```php
// Before: Broken syntax
if ($user->isActive()) {
dd($user); // ‚ùå Breaks the if statement
    return response()->json(['status' => 'active']);
}

// After: Context-aware insertion
if ($user->isActive()) {
    dd($user); // ‚úÖ Correctly placed inside the block
    return response()->json(['status' => 'active']);
}
```
